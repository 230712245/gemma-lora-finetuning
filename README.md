# Gemma LoRA Fine-Tuning

A generative AI project focused on fine-tuning and evaluating Google's Gemma 2B large language model (LLM) using LoRA (Low-Rank Adaptation).

## Features
- Large Language Model (LLM) fine-tuning
- Google Gemma 2B integration
- LoRA (Low-Rank Adaptation) training
- Parameter-efficient fine-tuning (PEFT)
- Prompt templating and tokenization
- Conversational chat formatting
- Style transfer for dialogue generation
- Token masking for loss computation
- Forward and backward pass training pipeline
- LLM-as-a-Judge evaluation workflow
- Custom evaluation metric implementation
- System-prompt-based scoring
- Cross entropy evaluation
- Quantitative benchmarking for generated responses
- Opik evaluation framework integration
- Generative AI experimentation

## Files
- `gemma_lora_finetuning.ipynb` → Jupyter Notebook

## Technologies
- Python
- Hugging Face Transformers
- Google Gemma 2B
- LoRA / PEFT
- Liquid AI LFM-40B
- Opik
- PyTorch / TensorFlow
- Jupyter Notebook

## Description
This project explores parameter-efficient large language model fine-tuning, conversational style transfer, and LLM evaluation workflows using Google's Gemma 2B model and LoRA-based adaptation techniques.
